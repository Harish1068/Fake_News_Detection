# Fake News Detection Project

This repository contains a notebook for Fake News Detection using machine learning techniques.

## Notebook View

The notebook file `FakeNews.ipynb` can appear as raw JSON on GitHub. To view the notebook rendered nicely, please use [nbviewer](https://nbviewer.org/github/Harish1068/Fake_News_Detection/blob/main/FakeNews.ipynb).

## Files

- `FakeNews.ipynb`: Jupyter notebook containing the code and analysis.
- `Fake.csv` and `True.csv`: Dataset files used in the notebook.

## How to Run

To run the notebook locally:

1. Install Jupyter Notebook: `pip install notebook`
2. Open the notebook: `jupyter notebook FakeNews.ipynb`
3. Run the cells sequentially.
